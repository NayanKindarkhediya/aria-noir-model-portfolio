# 🖤 Aria Noir – Model Portfolio Website

Aria Noir is a modern and elegant **fashion model portfolio website** designed to showcase professional modeling work with a luxurious black aesthetic. The website features stunning visuals, smooth animations, premium layouts, and a fully responsive design to create a high-end portfolio experience across all devices.

## ✨ Features

- 📱 Fully responsive design (Mobile, Tablet & Desktop)
- 🖤 Premium luxury black theme
- 🎯 Hero section with professional model showcase
- 📸 Portfolio & gallery section
- 👤 About model section
- 💼 Services / Brand collaborations
- 🌟 Featured photoshoot showcase
- 💬 Testimonials section
- 📞 Contact section
- 🧭 Modern navigation with smooth scrolling
- ✨ Interactive animations and transitions

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap
- JavaScript

## 🎯 Purpose of the Project

- Practice modern portfolio website development
- Showcase professional fashion modeling work
- Improve responsive frontend development skills
- Build a premium portfolio-ready website
- Create an elegant luxury web experience
